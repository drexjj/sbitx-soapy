# sbitx-soapy
Experimental hardware control layer software, Soapy-based FE scripts, and modules for sBitx Hardware. This is under active development so please be patient with us. 

Based on on work from Juan WP3DN, Dave N1AI, Mike KB2ML, JJ W9JES, and Rafael PU2UIT




Clone this repo with:
```console
cd ~ && git clone https://github.com/drexjj/sbitx-soapy && cd sbitx-soapy && ./install.sh
```

The run `startpihpsdr.py`
